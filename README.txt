-- SUMMARY --

Provides a custom scheme wrapper to store the images and call the images from a perticular directory
instead of public file directory defined.!

-- REQUIREMENTS --
None.

-- INSTALLATION --
Just Enable the module on your Drupal setup.

--How To USE---
When ever you try to use this module. just use chrome://  instead public:// The images will go and called from,
sites/default/files/chrome folder instead of common sites/default/files/
